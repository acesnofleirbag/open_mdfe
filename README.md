# OPEN SEFAZ

NodeJS package para emissão de NF-e, NFC-e e consulta de dados

### Requisitos

- NodeJS v18+

### Instalação

```bash
$ npm install @acesnofleirbag/open-sefaz
```

### Especificação

A especificação implementada encontra-se na pasta `/specs`
